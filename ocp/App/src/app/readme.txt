Nesse exemplo, utilizo o O (Open Closed Principle) do SOLID de forma que posso adicionar vários gêneros de games para a minha aplicação, pois ela é escalável, bastando apenas criar novas classes para esses gêneros (que é uma interface que pode ser implementadae classes), sem necessitar exatamente adicionar tudo em uma classe apenas. Ou seja, também respeita o S (Single Responsability Principle) do SOLID.
